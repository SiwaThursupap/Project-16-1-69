# โปรแกรมคำนวณค่าไฟฟ้า

โปรเจกต์นี้ใช้สำหรับคำนวณค่าไฟฟ้าตามการใช้พลังงานเป็นกิโลวัตต์-ชั่วโมง (kWh) โดยใช้อัตราค่าไฟแบบขั้นบันได (Tiered Pricing) และมีค่าบริการคงที่ 50 บาทต่อบิล

## โครงสร้างอัตราค่าไฟฟ้า

- **ค่าใช้พลังงาน (คิดตามช่วงหน่วย):**
  - 0–150 หน่วย: 3.50 บาท/หน่วย
  - 151–400 หน่วย: 4.20 บาท/หน่วย
  - 401 หน่วยขึ้นไป: 5.00 บาท/หน่วย
- **ค่าบริการ (คงที่):**
  - 50 บาท (คิดครั้งเดียวต่อบิล ไม่ขึ้นกับจำนวนหน่วยที่ใช้)

---

## Pseudocode (รหัสเทียม)

รหัสเทียมด้านล่างนี้อธิบายขั้นตอนการคำนวณค่าไฟฟ้ารวม.

```text
START

  PRINT "กรอกหน่วยการใช้ไฟฟ้า (kWh): "
  READ units

  SET serviceCharge = 50
  SET energyCharge = 0

  IF units <= 0 THEN
      energyCharge = 0
  ELSE
      // คำนวณค่าไฟแบบขั้นบันได
      IF units <= 150 THEN
          energyCharge = units * 3.50
      ELSE IF units <= 400 THEN
          // 150 หน่วยแรกคิด 3.50
          energyCharge = 150 * 3.50
          // หน่วยที่เหลือ (units - 150) คิด 4.20
          energyCharge = energyCharge + (units - 150) * 4.20
      ELSE
          // 150 หน่วยแรกคิด 3.50
          energyCharge = 150 * 3.50
          // 250 หน่วยถัดไป (151–400) คิด 4.20
          energyCharge = energyCharge + 250 * 4.20
          // หน่วยที่เกินจาก 400 ขึ้นไปคิด 5.00
          energyCharge = energyCharge + (units - 400) * 5.00
      END IF
  END IF

  // บวกค่าบริการคงที่
  totalBill = energyCharge + serviceCharge

  PRINT "จำนวนหน่วยที่ใช้      : " + units
  PRINT "ค่าใช้พลังงาน         : " + energyCharge
  PRINT "ค่าบริการ              : " + serviceCharge
  PRINT "ยอดรวมที่ต้องจ่าย      : " + totalBill

END
```

## หมายเหตุ

- การคิดราคาจะเป็นแบบ **ขั้นบันได**:
  - 150 หน่วยแรกคิดที่ 3.50 บาท/หน่วย เสมอ
  - 250 หน่วยถัดไป (151–400) คิดที่ 4.20 บาท/หน่วย
  - หน่วยที่เกิน 400 ขึ้นไป คิดที่ 5.00 บาท/หน่วย
- มีการคิด **ค่าบริการคงที่ 50 บาท** ทุกครั้ง แม้จะใช้ไฟ 0 หน่วยก็ตาม
